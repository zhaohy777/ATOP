# ATP
Code for Paper "ATP:Adversarial Topic-aware Prompt-tuning for Cross-topic Automated Essay Scoring"

### Requirements
```
python >= 3.7
torch >= 1.10.0
transformers >= 4.10.0
```

### Run
```
python experiments/cli.py --config_yaml soft_prompt.yaml 
```
